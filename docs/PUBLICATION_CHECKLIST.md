# Publication Checklist

Before pushing this to a public GitHub repo:

- [ ] Search for tokens, local keys, device IDs, emails, home maps, IPs you do not want public.
- [ ] Replace private Cast UUID/host defaults with placeholders.
- [ ] Make every machine path configurable.
- [ ] Confirm `.env`, logs, audio files, caches, and `~/.roborock` are not copied.
- [ ] Run tests and library validation.
- [ ] Decide public repo name, probably `dustlord` or `lord-dusk`.
- [ ] Add demo media only after visual/audio review.
